# Donghua Episode Tracker

A simple web app for tracking donghua watch progress across your favorite
shows.

## Features

- Google sign-in
- Add and manage donghua titles
- Track watched episode counts
- View latest update history
- Search, sort, and paginate shows
- See frequently used shows
- Dashboard stats for total shows, in-progress shows, watched episodes, and most-used show
- Import and export tracker data
- Light and dark mode
- Responsive design for desktop and mobile
