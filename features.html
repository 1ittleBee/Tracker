Tracker Features

Quick access guide for the main tools in Donghua Episode Tracker.

Back to Tracker

Add Shows

Add one or many donghua titles by entering each name on a new line.

Update Episodes

Use plus, minus, or edit episode controls to keep progress current.

Rename and Delete

Change a saved title name or remove shows you no longer track.

Latest History

See the most recent episode change for each donghua in your list.

Search and Sort

Find titles quickly and sort by creation time, name, or episode.

Pagination

Choose shows per page, move between pages, or jump to a page.

Frequently Used

Open the top updated shows list based on usage count.

Import and Export

Import JSON backups and export your tracker data as JSON or CSV.

Dark Mode

Switch the tracker interface between light and dark themes.
